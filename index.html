About me


Previous Next

My name is Vibha Seshadri (as you could probably tell by the large webpage title) and I am a rising senior at Saratoga High. I am part of my school's newspaper staff and speech and debate team. Additionally, I take Indian classical dance lessons outside of school and perform on a regular basis. I enjoy writing, sleeping, reading, and spending time outside. Thanks for visiting my page, and I hope you enjoy it!

City Scroller

We created a game in the first of Girls Who Code in which a bear travels through space near the moon while avoiding aliens and catching honey pots for points. We spent two days creating this and it was extremely rewarding to play the game after we finished it.

Splatter Art

In the first week of the program, we got acquainted with javascript through the Khan Academy coding interface. To experiment with mouse interaction we were instructed to create splatter art with code. My code allows the user to trace the screen, and have random circles of different color show up. Additionally, a mirror image of the splatter art shows up on the screen as well.

Fish Tank

Also on the Khan Academy interface, Era and I created a fish tank which would produce random fish every time the screen was clicked. This exercise also made use of mouse interaction.

Robot fun!

To learn python, we were given a day to play and code something creative with the robot. The group I was a part of programmed a robot to write the "Hello World" while playing the song "Hello, Goodbye" by The Beatles.

Obamicon

When working with robots, my partner Asila and I programmed a robot in python to take a picture of us and then change all the pixels, based on the color intensity of the RGB value, to make us look the same color as Obama in this hope poster.

Guitarduino

My group and I created an arduino project which resembles the video game guitar hero. We had four rows of LED's and when the last LED lit up, the player had to press the proper button. The two songs which could be played were "Don't stop believing" and "Twinkle, Twinkle little star"

Homepage 1.0

We began using HTML,CSS, and javascript to work on our Girls Who Code portfolios.I was immediatley enthralled by the project because I have always had a love for design, and now I got to combine with my love for code!"

Homepage 2.0 & 3.0

We continued working on our portfolio's at the beginning of the week. For homepage 2.0 we experimented with JQuery, and for homepage 3.0 we worked with bootstrap."

2048

We created a program which would emulate the popular game 2048. This game marked the beginning of the week we spent on covering data structures and algorithms. Creating the program was really fun because for two reasons. Firstly, I love writing algoritms. Secondly, 2048 is awesome!

Mapbox

To gain a better understanding of APIs and databases, we used Mapbox's API to create a map of our favorite restaurants in the bay area. We placed the information for these restaurants in a Google spreadsheet, and then used the spreadsheet as the database we used for the code.

Password 1.0

We used python to create a program which allows users to create a username and password and sign in using them. We did this by hashing the password and using the username as the key in the dictionary that held the hashed passwords.
